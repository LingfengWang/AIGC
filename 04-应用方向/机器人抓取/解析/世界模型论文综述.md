# 机器人抓取世界模型论文综述

## 一、研究背景与概述

### 1.1 什么是世界模型？

**世界模型 (World Model)** 是机器人智能的核心组件，让机器人能够：
- **理解物理世界**：理解物体之间的关系、物理规律
- **预测未来状态**：给定当前状态和动作，预测结果
- **想象和规划**：在脑海中模拟动作后果

```
世界模型 = 理解 + 预测 + 想象
```

### 1.2 为什么抓取需要世界模型？

**抓取**是机器人操作的核心任务，需要：
- 理解物体几何形状
- 预测物体可抓取性
- 理解物体之间关系（遮挡、相互影响）
- 想象抓取后的结果

### 1.3 核心挑战

1. **Sim-to-Real Gap**：模拟与现实的差距
2. **复杂场景**：多物体、遮挡、光照变化
3. **泛化能力**：对新物体、新场景的适应
4. **实时性**：需要快速推理

## 二、核心论文详解

### 2.1 Dreamer 系列 (DeepMind)

#### Dreamer (2020) - 基于模型的强化学习

| 项目 | 内容 |
|------|------|
| 论文 | Dreamer: Learning World Models for Sparse Reward RL |
| 作者 | Hafner et al., DeepMind |
| arXiv | 1912.05663 |

**核心创新**：
- 潜在空间 (Latent Space) 学习
- 图像重建 + 奖励预测
- 世界模型 + 想象规划

**架构**：
```
观测 → 编码器 → 潜在状态 z_t
           ↓
    潜在动力学模型 p(z_{t+1}|z_t, a_t)
           ↓
    奖励预测 r_t ~ p(r_t|z_t, a_t)
           ↓
    策略网络 π(a_t|z_t) + 价值网络 V(z_t)
```

**贡献**：
- 首次实现基于图像的世界模型端到端训练
- 在视觉控制任务上超越无模型方法

---

#### DreamerV3 (2023) - 通用世界模型

| 项目 | 内容 |
|------|------|
| 论文 | Mastering Diverse Domains through World Models |
| 作者 | Hafner et al., DeepMind |
| arXiv | 2303.10145 |

**核心创新**：
- **Symlog 变换**：处理不同尺度的奖励
- **自由运行 (Free-run)**：自回归生成
- **更大的模型容量**：更大参数量的 Transformer

**突破**：
- 首次在离散和连续动作空间中统一
- 适应不同奖励尺度的任务

### 2.3 ViVa (2026) - 视频生成价值模型

| 项目 | 内容 |
|------|------|
| 论文 | ViVa: A Video-Generative Value Model for Robot RL |
| 作者 | 多机构 |
| arXiv | 2604.01901 |

**核心创新**：
- **视频生成 + 价值评估** 联合
- 基于真实视频数据训练
- 预测长期累积奖励

**架构**：
```
观察 → 视频生成器 → 未来帧预测
                ↓
         价值模型 → 状态价值估计
                ↓
         策略优化 → 动作生成
```

**优势**：
- 更真实的未来预测
- 端到端可微训练

## 四、核心方法分类

### 4.1 隐式世界模型

- **特点**：不显式预测像素，而是学习潜在动力学
- **代表**：Dreamer, IRIS
- **优点**：计算高效、泛化好
- **缺点**：可能丢失细节

### 4.2 显式世界模型 (视频生成)

- **特点**：直接预测未来帧
- **代表**：ViVa, Veo-Act
- **优点**：真实、可解释
- **缺点**：计算量大

### 4.3 基于扩散的世界模型

- **特点**：使用扩散模型生成未来
- **代表**：近期工作
- **优点**：生成质量高
- **缺点**：采样慢
