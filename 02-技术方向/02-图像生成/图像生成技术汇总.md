### 3. 图像编辑 (Image Editing)

**编辑类型**:
```
1. 局部编辑

2. 全局编辑

   ├─ 颜色调整

3. 语义编辑
```

### 4. 可控生成 (Controlled Generation)

**控制方式**:
| 控制类型 | 输入 | 应用场景 |
|---------|------|---------|
| Canny | 边缘图 | 线稿上色 |
| Depth | 深度图 | 3D 感知生成 |
| Pose | 人体姿态 | 角色生成 |
| Scribble | 涂鸦 | 草图生成 |
| Segment | 语义分割 | 区域控制 |
| Reference | 参考图 | 角色一致性 |

**ControlNet 系列**:
```
ControlNet (2023)
├─ Canny (边缘检测)

├─ Inpaint (局部重绘)

```

```bash
  --instance_data_dir path/to/style_images \
  --output_dir path/to/output \
  --train_batch_size 1 \
  --learning_rate 1e-6 \
  --max_train_steps 1000
```

### 主流模型对比 (2026)

| 模型 | 分辨率 | 速度 | 质量 | 显存需求 | 协议 |
|------|--------|------|------|---------|------|

## 🔮 技术趋势

### 2025-2026 趋势

1. **流模型普及**
   - SD3、FLUX 采用流匹配
   - 采样速度更快
   - 训练更稳定

2. **架构融合**
   - VLM + Diffusion
   - 理解能力增强
   - 多模态统一

3. **一步生成**
   - 蒸馏技术进步
   - 实时生成成为可能
   - 移动端部署

4. **像素空间生成**
   - 无需 VAE 压缩
   - 细节更丰富
   - 训练成本降低

5. **自回归方法**
   - VAR、NextFlow
   - 新的生成范式
   - 质量媲美 Diffusion
