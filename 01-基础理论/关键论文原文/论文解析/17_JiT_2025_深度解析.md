## 核心贡献

### 3.1 统一框架

**核心贡献！**

```
JiT (Just in Time) 统一框架:

预测目标统一:

一般形式:
target = a·ε + b·x₀
```

## 数学原理

### 4.1 统一预测框架

```
统一预测目标:

加噪过程:
x_t = √ᾱ_t x₀ + √(1-ᾱ_t) ε

一般预测目标:
target = a_t · ε + b_t · x₀

其中 a_t, b_t 是时间的函数

特例:
1. DDPM (预测噪声):
   a_t = 1, b_t = 0
   target = ε

2. x₀-prediction:
   a_t = 0, b_t = 1
   target = x₀

3. v-prediction:
   a_t = √ᾱ_t, b_t = -√(1-ᾱ_t)
   target = √ᾱ_t ε - √(1-ᾱ_t) x₀
```

### 4.2 损失函数

```
统一损失:

L = E[||target_pred - target_true||²]

其中:
- target_pred: 模型预测
- target_true: 真实目标

梯度分析:

∂L/∂θ = 2(target_pred - target_true) · ∂target_pred/∂θ

关键:
- target_true 的方差影响梯度
- 噪声方差固定
- x₀ 方差变化
```

### 4.3 梯度稳定性分析

```
梯度方差:

预测噪声:
Var[∂L/∂θ] ∝ Var[ε] = 1 (固定)

预测 x₀:
Var[∂L/∂θ] ∝ Var[x₀] (变化大)

预测 v:
Var[∂L/∂θ] ∝ Var[v] (介于两者之间)

结论:
- 预测噪声梯度最稳定
- 预测 x₀ 梯度最不稳定
- 预测 v 折中
```

## 训练方法

### 6.1 统一训练算法

```
Algorithm 1: JiT 统一训练

Input: 数据集 {x₀⁽¹⁾, ..., x₀⁽ᴺ⁾}, 预测目标系数 (a, b)
Initialize: UNet 参数 θ

repeat
    for each x₀⁽ⁱ⁾:
        # 采样时间
        t ~ Uniform(1, T)
        
        # 采样噪声
        ε ~ N(0, I)
        
        # 加噪
        x_t = √ᾱ_t x₀⁽ⁱ⁾ + √(1-ᾱ_t) ε
        
        # 计算真实目标
        target_true = a_t · ε + b_t · x₀⁽ⁱ⁾
        
        # 预测
        target_pred = UNet(x_t, t)
        
        # 损失
        L = ||target_pred - target_true||²
    
    更新 θ
until 收敛
```

## 代码实现

### 8.1 统一预测目标

```python
class UnifiedDiffusion:
    def __init__(self, target_type='noise'):
        self.target_type = target_type
    
    def get_target(self, x0, noise, t):
        """获取预测目标"""
        # 计算 alpha_cumprod
        alpha_cumprod = self.alphas_cumprod[t]
        
        if self.target_type == 'noise':
            # DDPM: 预测噪声
            target = noise
        
        elif self.target_type == 'x0':
            # 预测原图
            target = x0
        
        elif self.target_type == 'v':
            # 预测速度
            sqrt_alpha = torch.sqrt(alpha_cumprod)
            sqrt_one_minus_alpha = torch.sqrt(1 - alpha_cumprod)
            target = sqrt_alpha * noise - sqrt_one_minus_alpha * x0
        
        return target
```

### 8.2 训练循环

```python
def train_jit(model, dataloader, target_type='noise', epochs=10):
    diffusion = UnifiedDiffusion(target_type)
    
    for epoch in range(epochs):
        total_loss = 0
        
        for x0, _ in dataloader:
            # 采样时间和噪声
            t = torch.randint(0, 1000, (x0.shape[0],))
            noise = torch.randn_like(x0)
            
            # 加噪
            xt = diffusion.add_noise(x0, noise, t)
            
            # 获取目标
            target = diffusion.get_target(x0, noise, t)
            
            # 预测
            pred = model(xt, t)
            
            # 损失
            loss = F.mse_loss(pred, target)
            
            # 反向传播
            loss.backward()
            
            # 梯度裁剪 (预测 x0 时需要)
            if target_type == 'x0':
                torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
        
        avg_loss = total_loss / len(dataloader)
        print(f'Epoch {epoch+1}: Loss = {avg_loss:.4f}')
```

## 深入理解

### 9.1 为什么预测噪声最优？

```
理论解释 1: 梯度稳定性

预测噪声:
- 噪声分布固定 N(0,I)
- 梯度方差稳定
- 训练稳定

预测 x₀:
- x₀ 分布复杂
- 梯度方差大
- 训练不稳定

预测 v:
- 介于两者之间
- 折中
```

```
理论解释 2: 信噪比

预测噪声:
- 目标是纯噪声
- 简单
- 易学习

预测 x₀:
- 目标是复杂数据
- 困难
- 难学习
```

### 9.2 何时考虑其他目标？

```
预测 x₀ 的场景:
- 需要直接重建
- 某些特殊任务
- 不推荐

预测 v 的场景:
- 需要快速采样
- 流匹配场景
- 折中选择

推荐:
- 默认预测噪声
- 特殊情况再考虑其他
```
