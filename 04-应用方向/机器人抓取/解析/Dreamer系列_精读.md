## Dreamer: Learning World Models for Sparse Reward RL (2020)

### 基本信息

| 项目 | 内容 |
|------|------|
| 论文标题 | Dreamer: Learning World Models for Sparse Reward RL |
| 作者 | Danijar Hafner, Timothy Lillicrap, Jimmy Ba, Mohammad Norouzi |
| 机构 | DeepMind, University of Toronto |
| 发表 | NeurIPS 2020 |
| arXiv | 1912.05663 |

---

### 核心问题

1. **稀疏奖励问题**：大多数状态下没有即时反馈
2. **样本效率**：现实世界交互成本高
3. **想象力规划**：如何在脑海中模拟未来？

---

### 核心创新

**三大组件**：

```
┌─────────────────────────────────────────────────────────────┐
│                    Dreamer 架构                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 世界模型 (World Model)                                  │
│     观测 → 编码 → 潜在状态 z                                │
│     z_t, a_t → 潜在动力学 p(z_{t+1}|z_t, a_t)              │
│     → 奖励预测 r_t ~ p(r_t|z_t, a_t)                        │
│                                                              │
│  2. 演员-评论家 (Actor-Critic)                              │
│     潜在状态 → 策略 π(a_t|z_t)                              │
│     潜在状态 → 价值 V(z_t)                                  │
│                                                              │
│  3. 想象 rollout (Dream)                                    │
│     从潜在状态开始，想象未来轨迹                             │
│     通过 imagined rewards 训练策略                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

---

### 技术细节

#### 1. 潜在世界模型

**编码器**：
```python
def encoder(o_t):
    # o_t: 图像观测
    h_t = gru(o_t)  # RNN 提取序列信息
    z_t = q(z|h)   # 潜在变量 (离散或连续)
```

**动力学模型**：
```python
def dynamics(z_t, a_t):
    # 预测下一时刻潜在状态
    z_{t+1} = model(z_t, a_t)
    # 预测奖励
    r_t = reward_model(z_t, a_t)
```

#### 2. 损失函数

```
L = L_recon + β * L_kl + L_reward + L_dis
```

- **重建损失**：重构图像
- **KL 散度**：正则化潜在空间
- **奖励预测**：预测即时奖励
- **判别器损失**：区分真实/生成

#### 3. 想象 rollout

```python
def imagine_trajectory(z_0, policy):
    """从初始状态想象未来"""
    z = z_0
    for t in range(horizon):
        a = policy(z)          # 根据当前状态选择动作
        z = dynamics(z, a)     # 预测下一状态
        r = reward(z)          # 预测奖励
        yield z, a, r
```

### 关键洞察

1. **潜在空间优势**：
   - 高维图像 → 低维潜在空间
   - 更快的规划
   - 更好的泛化

2. **想象规划**：
   - 不需要真实环境交互
   - 可以快速探索
   - 减少样本需求

3. **端到端训练**：
   - 世界模型 + 策略联合优化
   - 相互促进

## DreamerV3: Mastering Diverse Domains (2023)

### 核心改进

| 改进点 | Dreamer | DreamerV3 |
|--------|---------|-----------|
| 奖励处理 | 归一化 | **Symlog 变换** |
| 动作空间 | 连续/离散分别处理 | **统一处理** |
| 模型规模 | 较小 | **更大容量** |
| 应用范围 | 单一领域 | **多领域** |

---

### Symlog 变换

**问题**：不同任务的奖励尺度差异巨大

```
任务A: 奖励 0.001 ~ 0.01
任务B: 奖励 -100 ~ 100
任务C: 奖励 0/1 (二元)
```

**解决方案**：Symlog

```python
def symlog(x):
    """对称对数变换"""
    return np.sign(x) * np.log1p(np.abs(x))

def symexp(x):
    """逆变换"""
    return np.sign(x) * (np.exp(np.abs(x)) - 1)
```

**效果**：
- 自动适应不同奖励尺度
- 无需人工调参

## 面试要点

### Q1: Dreamer 核心思想？
通过学习潜在世界模型，在潜在空间进行想象规划，大幅提升样本效率。

### Q2: 为什么要用潜在空间？
- 降低计算复杂度
- 去除冗余视觉信息
- 更适合规划和控制

### Q3: DreamerV3 的改进？
- Symlog 处理不同尺度奖励
- 统一处理离散/连续动作
- 更大容量模型

### Q4: 世界模型 vs 无模型？
- 世界模型：样本效率高，但有模型误差
- 无模型：泛化好，但样本效率低
- 两者可结合 (MBRL)

### Q5: 想象 rollout 如何工作？
- 从当前状态开始
- 用学到的动力学模型预测未来
- 用预测的奖励训练策略
- 无需真实环境交互
