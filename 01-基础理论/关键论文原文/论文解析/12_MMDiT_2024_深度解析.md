## 数学原理

### 4.1 多模态注意力

```
联合注意力机制:

输入:
- 文本 tokens: T = [t₁, t₂, ..., t_N]
- 图像 tokens: I = [i₁, i₂, ..., i_M]

拼接:
X = [T; I] = [t₁, ..., t_N, i₁, ..., i_M]

自注意力:
Attention(X) = softmax(QK^T/√d)V

其中:
Q = W_Q · X
K = W_K · X
V = W_V · X

关键:
- 文本和图像在同一空间
- 可以互相 Attention
- 信息自由流动
```

### 4.2 流匹配目标

```
Flow Matching 损失:

L = E[||v_θ(x_t, t, y) - (x₁ - x₀)||²]

其中:
- x₀: 噪声
- x₁: 数据
- x_t: (1-t)x₀ + tx₁ (直线路径)
- y: 文本条件
- v_θ: 预测的速度场

与 Diffusion 对比:
- Diffusion: 预测噪声 ε
- Flow Matching: 预测速度 v
- 更直接的目标
```

### 4.3 条件注入

```
文本条件处理:

传统 Cross-Attention:
Q = 图像特征
K, V = 文本特征

MMDiT:
- 文本和图像平等
- 联合自注意力
- 双向交互

优势:
- 更好的对齐
- 更深入的理解
- 文字渲染更好
```

### 5.2 Transformer Block

```
MMDiT Block:

输入：文本 tokens T, 图像 tokens I

1. 文本自注意力
   T' = SelfAttn(T)

2. 图像自注意力
   I' = SelfAttn(I)

3. 联合注意力
   [T''; I''] = JointAttn([T'; I'])

4. 文本 MLP
   T''' = MLP(T'')

5. 图像 MLP
   I''' = MLP(I''')

输出：T''', I'''

特点:
- 文本和图像分别处理
- 联合注意力融合
- 平等的信息流
```

### 5.3 与 SDXL 对比

```
SDXL (UNet):
┌─────────────┐
│ 文本 → CrossAttn → UNet → 图像 │
└─────────────┘
文本是条件，图像是主体

MMDiT:
┌─────────────┐
│ 文本 → JointAttn ← 图像 │
│     ↓             ↓     │
│  文本塔        图像塔    │
└─────────────┘
文本和图像平等
```

## 训练方法

### 6.1 训练算法

```
Algorithm 1: MMDiT 训练

Input: 图像 + 文本数据集 {(x⁽¹⁾, y⁽¹⁾), ...}
Initialize: MMDiT 参数 θ

repeat
    for each (x⁽¹⁾, y⁽¹⁾):
        # 文本编码
        T = TextEmbed(y⁽¹⁾)
        
        # 图像 Patchify
        I = PatchEmbed(x⁽¹⁾)
        
        # 随机时间步
        t ~ Uniform(0, 1)
        
        # 噪声和数据
        x₀ ~ N(0, I)
        x₁ = I (图像 tokens)
        
        # 直线路径
        x_t = (1-t)x₀ + tx₁
        
        # 预测速度场
        v_θ = MMDiT(x_t, t, T)
        
        # 目标
        u = x₁ - x₀
        
        # 损失
        L = ||v_θ - u||²
    
    # 反向传播
    ∇_θ L = Backprop(L)
    θ ← θ - η ∇_θ L
    
until 收敛
```

## 代码实现

### 8.1 MMDiT Block

```python
class MMDiTBlock(nn.Module):
    def __init__(self, dim, num_heads, mlp_ratio=4.0):
        super().__init__()
        
        # 文本自注意力
        self.text_self_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        self.text_norm1 = nn.LayerNorm(dim)
        
        # 图像自注意力
        self.image_self_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        self.image_norm1 = nn.LayerNorm(dim)
        
        # 联合注意力
        self.joint_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        self.joint_norm = nn.LayerNorm(dim * 2)
        
        # MLP
        self.text_mlp = nn.Sequential(
            nn.Linear(dim, int(dim * mlp_ratio)),
            nn.GELU(),
            nn.Linear(int(dim * mlp_ratio), dim),
        )
        self.text_norm2 = nn.LayerNorm(dim)
        
        self.image_mlp = nn.Sequential(
            nn.Linear(dim, int(dim * mlp_ratio)),
            nn.GELU(),
            nn.Linear(int(dim * mlp_ratio), dim),
        )
        self.image_norm2 = nn.LayerNorm(dim)
    
    def forward(self, text, image):
        # 文本自注意力
        text_norm = self.text_norm1(text)
        text_attn = self.text_self_attn(text_norm, text_norm, text_norm)[0]
        text = text + text_attn
        
        # 图像自注意力
        image_norm = self.image_norm1(image)
        image_attn = self.image_self_attn(image_norm, image_norm, image_norm)[0]
        image = image + image_attn
        
        # 联合注意力
        combined = torch.cat([text, image], dim=1)
        combined_norm = self.joint_norm(combined)
        combined_attn = self.joint_attn(combined_norm, combined_norm, combined_norm)[0]
        text_attn, image_attn = combined_attn.split([text.shape[1], image.shape[1]], dim=1)
        
        text = text + text_attn
        image = image + image_attn
        
        # MLP
        text = text + self.text_mlp(self.text_norm2(text))
        image = image + self.image_mlp(self.image_norm2(image))
        
        return text, image
```

### 8.2 MMDiT 模型

```python
class MMDiT(nn.Module):
    def __init__(self, text_dim=1024, image_dim=1024, depth=24, num_heads=16):
        super().__init__()
        
        # 文本嵌入
        self.text_embed = nn.Linear(768, text_dim)  # CLIP 输出 768
        
        # 图像 Patch 嵌入
        self.patch_embed = PatchEmbed(patch_size=2, embed_dim=image_dim)
        
        # 时间嵌入
        self.time_embed = nn.Sequential(
            nn.Linear(1, image_dim * 4),
            nn.SiLU(),
            nn.Linear(image_dim * 4, image_dim * 6),  # 用于 adaLN
        )
        
        # Transformer Blocks
        self.blocks = nn.ModuleList([
            MMDiTBlock(image_dim, num_heads) for _ in range(depth)
        ])
        
        # 输出投影
        self.norm = nn.LayerNorm(image_dim)
        self.out_proj = nn.Linear(image_dim, 4)  # VAE latent 通道
        
        self.image_dim = image_dim
    
    def forward(self, x, t, text):
        # 文本嵌入
        text = self.text_embed(text)  # (B, N_text, D)
        
        # 图像 Patchify
        image = self.patch_embed(x)  # (B, N_image, D)
        
        # 时间嵌入
        t_emb = self.time_embed(t.float().unsqueeze(-1))
        
        # Transformer Blocks
        for block in self.blocks:
            text, image = block(text, image)
        
        # 输出
        image = self.norm(image)
        image = self.out_proj(image)
        
        # Unpatchify
        image = self.unpatchify(image)
        
        return image  # 预测的速度场
```

### 8.3 训练循环

```python
def train_mmdit(model, dataloader, optimizer, device, epochs=10):
    model.train()
    
    for epoch in range(epochs):
        total_loss = 0
        
        for batch_idx, (images, texts) in enumerate(dataloader):
            images, texts = images.to(device), texts.to(device)
            optimizer.zero_grad()
            
            # 文本编码
            text_embeds = clip_encode(texts)  # (B, 77, 768)
            
            # 图像 Patchify
            image_tokens = model.patch_embed(images)  # (B, N, D)
            
            # 随机时间步
            t = torch.rand(images.shape[0], device=device)
            
            # 噪声和数据
            x0 = torch.randn_like(image_tokens)
            x1 = image_tokens
            
            # 直线路径
            xt = (1 - t[:, None, None]) * x0 + t[:, None, None] * x1
            
            # 预测速度场
            v_pred = model(xt, t, text_embeds)
            
            # 目标
            u = x1 - x0
            
            # 损失
            loss = F.mse_loss(v_pred, u)
            
            # 反向传播
            loss.backward()
            optimizer.step()
            
            total_loss += loss.item()
        
        avg_loss = total_loss / len(dataloader)
        print(f'Epoch {epoch+1}: Loss = {avg_loss:.4f}')
```

### 8.4 采样函数

```python
@torch.no_grad()
def sample_mmdit(model, text_encoder, prompt, num_steps=28):
    """MMDiT 采样"""
    model.eval()
    
    # 文本编码
    text = text_encoder(prompt)  # (1, 77, 768)
    text = model.text_embed(text)
    
    # 从噪声开始
    x = torch.randn(1, 4, 128, 128).to(model.device)
    
    # Euler 积分
    dt = 1.0 / num_steps
    
    for i in range(num_steps):
        t = torch.full((1,), i * dt, device=model.device)
        
        # Patchify
        x_tokens = model.patch_embed(x)
        
        # 预测速度场
        v = model(x_tokens, t, text)
        
        # Unpatchify
        v = model.unpatchify(v)
        
        # Euler 步
        x = x + v * dt
    
    return x
```
