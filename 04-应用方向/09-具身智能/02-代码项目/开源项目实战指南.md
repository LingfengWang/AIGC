## 📦 项目分类

### 入门级项目 (⭐⭐)

#### 1. EVERY-EMBODIED - 从零构建具身智能机器人
- **Stars**: 1.4k
- **难度**: ⭐⭐
- **时间**: 2-4 小时
- **链接**: https://github.com/every-embodied/every-embodied

```bash
# 克隆项目
git clone https://github.com/every-embodied/every-embodied
cd every-embodied

# 安装依赖
pip install -e .

# 运行示例
python examples/hello_world.py
python examples/vla_inference.py

# 学习路径
# 1. 理解 VLA 模型基础架构
# 2. 运行预训练模型推理
# 3. 尝试简单任务
```

**学习收获**:
- 理解 VLA 模型基本原理
- 掌握具身智能推理流程
- 了解任务执行框架

---

#### 2. LINGBOT-VLA - 实用 VLA 基础模型
- **Stars**: 1k
- **难度**: ⭐⭐
- **时间**: 3-5 小时
- **链接**: https://github.com/lingbot-vla/lingbot-vla

```bash
# 克隆项目
git clone https://github.com/lingbot-vla/lingbot-vla
cd lingbot-vla

# 安装依赖
pip install -r requirements.txt

# 下载预训练模型
python scripts/download_model.py

# 运行推理示例
python examples/pick_and_place.py
python examples/navigation.py

# 尝试微调
python train/finetune.py --config configs/my_task.yaml
```

**学习收获**:
- 掌握 VLA 模型微调方法
- 理解任务特定适配
- 学习数据准备流程

#### 4. SIMPLERENV - 真实策略复现
- **Stars**: 1k
- **难度**: ⭐⭐⭐
- **时间**: 1-2 天
- **链接**: https://github.com/simpler-env/simpler-env

```bash
# 克隆项目
git clone https://github.com/simpler-env/simpler-env
cd simpler-env

# 安装依赖
pip install -e .

# 复现 RT-1 策略
python simpler_env/inference/rt1_inference.py \
  --task pick_up_fork \
  --ckpt_path checkpoints/rt1_checkpoint

# 评估策略性能
python simpler_env/evaluation/evaluate_policy.py \
  --task_suite kitchen \
  --policy rt1

# 可视化结果
python scripts/visualize_results.py --run_id xxx
```

**学习收获**:
- 理解 RT-1 等经典策略
- 掌握 Sim-to-Real 评估方法
- 学习策略复现技巧

### 高级项目 (⭐⭐⭐⭐)

#### 5. ManiSkill - GPU 并行化机器人模拟器
- **Stars**: 2.8k
- **难度**: ⭐⭐⭐⭐
- **时间**: 3-5 天
- **链接**: https://github.com/haosulab/ManiSkill

```bash
# 克隆项目
git clone https://github.com/haosulab/ManiSkill
cd ManiSkill

# 安装依赖 (需要 GPU)
pip install -e .

# 运行示例环境
python examples/maniskill2_envs/pick_cube.py

# 强化学习训练示例
python scripts/rl/train_sb3.py \
  --env_name PickCube-v1 \
  --algo sac \
  --total_timesteps 1000000

# 并行环境训练
python scripts/rl/train_sb3.py \
  --env_name PickCube-v1 \
  --num_envs 64 \
  --algo sac

# 自定义任务
# 参考 docs/tutorials/create_custom_env.md
```

**学习收获**:
- 掌握 GPU 加速的机器人仿真
- 理解强化学习训练流程
- 学习自定义任务创建

#### 6. DORA - 数据流机器人架构
- **Stars**: 3.5k
- **难度**: ⭐⭐⭐⭐
- **时间**: 3-5 天
- **链接**: https://github.com/dora-rs/dora

```bash
# 克隆项目
git clone https://github.com/dora-rs/dora
cd dora

# 安装依赖
cargo install dora-cli  # 或使用 pip
pip install dora-rs

# 运行示例
dora up
dora init my-project
cd my-project

# 创建数据流图
# 参考 examples/ 目录

# 运行机器人任务
dora build
dora run

# 分布式部署
dora start --daemon
```

**学习收获**:
- 理解数据流机器人架构
- 掌握低延迟数据流处理
- 学习分布式系统部署

#### 7. RLINF - 强化学习基础设施
- **Stars**: 3.1k
- **难度**: ⭐⭐⭐⭐
- **时间**: 3-5 天
- **链接**: https://github.com/rl-infra/rlinf

```bash
# 克隆项目
git clone https://github.com/rl-infra/rlinf
cd rlinf

# 安装依赖
pip install -e .

# 快速开始
python examples/quick_start.py

# 训练具身策略
rlinf train \
  --env embodied_nav \
  --algo ppo \
  --config configs/embodied_ppo.yaml

# 多 GPU 训练
rlinf train \
  --env embodied_nav \
  --num-gpus 4 \
  --deepspeed

# 可视化和监控
rlink dashboard --run-id xxx
```

**学习收获**:
- 掌握大规模 RL 训练
- 理解分布式训练架构
- 学习训练监控和调试

## 🔧 常见问题

### Q1: GPU 显存不足怎么办？
```bash
# 解决方案:
# 1. 使用梯度累积
python train.py --gradient_accumulation_steps 4

# 2. 混合精度训练
python train.py --fp16

# 3. 减小 batch size
python train.py --batch_size 8

# 4. 使用 DeepSpeed ZeRO
python train.py --deepspeed --zero_stage 2
```

### Q2: Sim-to-Real 差距大怎么办？
```python
# 解决方案:
# 1. 域随机化
# - 随机化光照、纹理、物体位置
# 2. 数据增强
# - 添加噪声、模糊、颜色扰动
# 3. 逐步迁移
# - 先在仿真中训练好，再在真实世界微调
# 4. 使用世界模型生成逼真数据
```

### Q3: 训练不稳定怎么办？
```python
# 解决方案:
# 1. 调整学习率
# - 使用学习率 warmup
# - 尝试更小的学习率
# 2. 梯度裁剪
torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)

# 3. 奖励缩放
# - 标准化奖励
# - 使用 reward clipping

# 4. 增加训练数据
# - 收集更多演示
# - 数据增强
```
