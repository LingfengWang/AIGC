## 研究背景

### 1.1 DDPM 的局限

DDPM (2020) 虽然成功，但存在明显问题：

| 问题 | 表现 | 影响 |
|------|------|------|
| **采样慢** | 需要 1000 步去噪 | 生成一张图需要几分钟 |
| **马尔可夫链** | 必须一步步采样 | 无法跳步，无法加速 |
| **随机性** | 每次采样结果不同 | 难以复现和调试 |

### 1.2 实际应用需求

```
实际应用场景对速度的要求:

交互式应用：
- 实时生成：< 1 秒
- 交互式编辑：< 5 秒
- 批量生成：尽可能快

DDPM 的问题:
- 1000 步 × 0.1 秒/步 = 100 秒/张
- 完全无法满足实际需求
```

### 1.3 理论动机

**核心观察**: DDPM 的训练目标不依赖于马尔可夫假设！

```
DDPM 的训练目标:
L = E[||ε - ε_θ(x_t, t)||²]

这个目标只要求:
1. q(x_t | x_0) 是正确的 (边缘分布)
2. 模型能预测噪声 ε

不要求:
- q(x_t | x_{t-1}) 必须是马尔可夫的
- 反向过程必须一步步来

洞察:
如果可以构造非马尔可夫的前向过程，
同时保持相同的边缘分布 q(x_t | x_0)，
那么训练目标不变，但可以加速采样！
```

## 核心问题

### 2.1 核心挑战

**如何加速 Diffusion 模型的采样过程，同时保持生成质量？**

### 2.2 关键问题

```
问题 1: 能否减少采样步数？
- DDPM: 1000 步 → DDIM: 50-100 步？

问题 2: 能否跳步采样？
- 从 x_t 直接到 x_{t-Δ}，跳过中间步骤？

问题 3: 能否确定性采样？
- 相同起点 → 相同结果，便于调试？
```

### 2.3 解决思路

```
核心思路: 重新定义扩散过程

DDPM:
- 前向：马尔可夫链 x₀ → x₁ → x₂ → ... → x_T
- 反向：必须一步步 x_T → x_{T-1} → ... → x₀

DDIM:
- 前向：非马尔可夫，但保持相同的边缘分布
- 反向：可以跳步 x_T → x_{T-Δ} → ... → x₀
```

## 核心贡献

### 3.1 非马尔可夫扩散过程

**这是 DDIM 最核心的贡献！**

```
DDPM 的前向过程 (马尔可夫):
q(x_t | x_{t-1}) = N(x_t; √(1-β_t)x_{t-1}, β_t I)

DDIM 的前向过程 (非马尔可夫):
q_σ(x_t | x_{t-1}, x_0) = N(x_t; √ᾱ_t x_0 + √(1-ᾱ_t - σ_t²)x_{t-1}, σ_t² I)

关键:
- 保持相同的边缘分布 q(x_t | x_0)
- 但条件分布 q(x_t | x_{t-1}, x_0) 可以不同
- σ_t 控制随机性程度
```

### 3.2 统一框架

```
DDIM 提出一个统一框架，包含 DDPM 作为特例:

当 σ_t = √((1-ᾱ_{t-1})/(1-ᾱ_t)) × √(1-ᾱ_t/ᾱ_{t-1}) × β_t 时:
→ 退化为 DDPM (随机采样)

当 σ_t = 0 时:
→ DDIM (确定性采样)

通过调节 σ_t，可以在随机性和确定性之间插值
```

### 3.3 加速采样

```
跳步采样策略:

原始: [1, 2, 3, ..., 999, 1000] (1000 步)
跳步: [1, 50, 100, ..., 950, 1000] (20 步)

效果:
- 速度提升 50 倍
- 质量略有下降但可接受
- FID: 9.4 → 12.5 (CIFAR-10)
```

### 3.4 确定性采样

```
当 σ_t = 0 时，采样是确定性的:

x_{t-1} = √ᾱ_{t-1} (x_t - √(1-ᾱ_t) ε_θ(x_t)) / √ᾱ_t
        + √(1-ᾱ_{t-1}) ε_θ(x_t)

好处:
1. 可复现：相同种子 → 相同结果
2. 可优化：可以直接优化 x_T 获得更好结果
3. 可插值：在 x_T 空间插值更平滑
```

## 数学原理

### 4.1 变分下界重推导

```
DDPM 的 ELBO:
L = E_q[log p(x_0) - log q(x_1:T | x_0)]

DDIM 的关键洞察:
这个下界不依赖于 q(x_1:T | x_0) 的具体形式
只要求边缘分布 q(x_t | x_0) 正确

因此可以重新定义联合分布:
q_σ(x_1:T | x_0) = q(x_T | x_0) × ∏_{t=2}^T q_σ(x_{t-1} | x_t, x_0)

其中 q_σ(x_{t-1} | x_t, x_0) 是新的条件分布
```

### 4.2 参数化

```
DDIM 的参数化:

q_σ(x_t | x_{t-1}, x_0) = N(x_t; √ᾱ_t x_0 + √(1-ᾱ_t - σ_t²)x_{t-1}, σ_t² I)

其中 σ_t 是自由参数，控制随机性

边缘分布保持不变:
q(x_t | x_0) = N(x_t; √ᾱ_t x_0, (1-ᾱ_t)I)
```

### 4.3 反向过程

```
学习反向分布:
p_θ(x_{t-1} | x_t) = N(x_{t-1}; μ_θ(x_t, t), σ_t² I)

其中均值:
μ_θ(x_t, t) = √ᾱ_{t-1} (x_t - √(1-ᾱ_t) ε_θ(x_t)) / √ᾱ_t
            + √(1-ᾱ_{t-1}) ε_θ(x_t)

方差 σ_t² 可以固定为:
σ_t² = ((1-ᾱ_{t-1})/(1-ᾱ_t)) × (1-ᾱ_t/ᾱ_{t-1}) × β_t
```

### 4.4 跳步采样

```
从序列 {1, 2, ..., T} 选择子序列 {τ_1, τ_2, ..., τ_S}

其中 S << T，例如:
- T = 1000, S = 50
- τ = [1, 20, 40, ..., 980, 1000]

采样时只在选定的时间步之间跳转:
x_{τ_{s-1}} = √ᾱ_{τ_{s-1}} (x_{τ_s} - √(1-ᾱ_{τ_s}) ε_θ(x_{τ_s})) / √ᾱ_{τ_s}
            + √(1-ᾱ_{τ_{s-1}}) ε_θ(x_{τ_s})
```

## 模型架构

### 5.1 与 DDPM 的关系

```
DDIM 与 DDPM 使用相同的模型架构!

模型: UNet
输入: x_t + t
输出: ε_θ(x_t, t)

区别:
- DDPM: 训练相同，采样不同
- DDIM: 可以直接使用 DDPM 训练好的模型
```

### 5.2 采样架构对比

```
DDPM 采样:
x_T → [去噪] → x_{T-1} → [去噪] → ... → x_0
       1000 次前向传播

DDIM 采样:
x_T → [去噪] → x_{T-Δ} → [去噪] → ... → x_0
       50 次前向传播 (Δ = 20)

节省: 95% 的计算量
```

## 训练方法

### 6.1 训练算法

```
Algorithm 1: DDIM 训练 (与 DDPM 相同)

Input: 数据集 {x₀⁽¹⁾, ..., x₀⁽ᴺ⁾}, T, 学习率 η
Initialize: UNet 参数 θ

repeat
    # 采样一个 batch
    {x₀⁽¹⁾, ..., x₀⁽ᴹ⁾} ~ 数据集
    
    for each x₀⁽ⁱ⁾ in batch:
        # 随机采样时间步
        t ~ Uniform({1, ..., T})
        
        # 采样噪声
        ε ~ N(0, I)
        
        # 计算带噪图像
        x_t = √ᾱ_t x₀⁽ⁱ⁾ + √(1-ᾱ_t) ε
        
        # 预测噪声
        ε_θ = UNet(x_t, t)
        
        # 计算损失
        L = ||ε - ε_θ||²
    
    # 反向传播
    ∇_θ L = Backprop(L)
    
    # 更新参数
    θ ← θ - η ∇_θ L
    
until 收敛

关键: DDIM 训练与 DDPM 完全相同!
```

### 6.2 采样算法

```
Algorithm 2: DDIM 采样

Input: 训练好的 UNet 参数 θ, 时间步序列 {τ_S, ..., τ_1}
Output: 生成的图像 x₀

# 从纯噪声开始
x_{τ_S} ~ N(0, I)

for s = S, S-1, ..., 1:
    # 预测噪声
    ε_θ = UNet(x_{τ_s}, τ_s)
    
    # 计算 x_{τ_{s-1}} (如果 s > 1)
    if s > 1:
        x_{τ_{s-1}} = √ᾱ_{τ_{s-1}} (x_{τ_s} - √(1-ᾱ_{τ_s}) ε_θ) / √ᾱ_{τ_s}
                    + √(1-ᾱ_{τ_{s-1}}) ε_θ
    else:
        x_0 = √ᾱ_0 (x_{τ_1} - √(1-ᾱ_{τ_1}) ε_θ) / √ᾱ_{τ_1}
            + √(1-ᾱ_0) ε_θ

return x₀
```

### 6.3 时间步选择策略

```
均匀采样:
τ = [1, T/S, 2T/S, ..., T]

简单，但可能不是最优

非均匀采样:
τ = [1, 5, 10, 20, 30, ..., T]

早期更密集，后期更稀疏
效果更好，因为早期去噪更关键
```

| σ_t | FID | 多样性 | 可复现性 |
|-----|-----|--------|---------|

### 7.3 隐空间插值

```
DDPM 插值:
在 x_T 空间插值 → 生成结果跳跃较大

DDIM 插值:
在 x_T 空间插值 → 生成结果平滑过渡

原因:
DDIM 是确定性的，x_T 与 x_0 有一一对应关系
```

### 7.4 图像修复应用

```
DDIM 可以用于图像修复 (Inpainting):

方法:
1. 已知区域保持不变
2. 未知区域从噪声开始
3. 交替更新已知和未知区域

效果:
- 修复结果自然
- 与已知区域无缝衔接
- 可以生成多种可能结果
```

## 代码实现

### 8.1 DDIM 采样器

```python
class DDIMScheduler:
    def __init__(self, T=1000, beta_start=1e-4, beta_end=0.02):
        self.T = T
        self.betas = torch.linspace(beta_start, beta_end, T)
        self.alphas = 1. - self.betas
        self.alphas_cumprod = torch.cumprod(self.alphas, dim=0)
    
    def set_timesteps(self, num_inference_steps):
        """设置推理步数"""
        self.num_steps = num_inference_steps
        # 均匀采样时间步
        self.timesteps = torch.linspace(0, self.T-1, num_inference_steps).long()
    
    def step(self, noise_pred, t, x_t):
        """DDIM 采样一步"""
        # 获取当前和下一步的 alpha_cumprod
        alpha_prod_t = self.alphas_cumprod[t]
        alpha_prod_t_prev = self.alphas_cumprod[t - 1] if t > 0 else torch.ones_like(alpha_prod_t)
        
        # 计算 x_0 的预测
        pred_original_sample = (x_t - (1 - alpha_prod_t).sqrt() * noise_pred) / alpha_prod_t.sqrt()
        
        # 计算 x_{t-1}
        pred_sample_direction = (1 - alpha_prod_t_prev).sqrt() * noise_pred
        x_t_prev = alpha_prod_t_prev.sqrt() * pred_original_sample + pred_sample_direction
        
        return x_t_prev
```

### 8.2 加速采样

```python
@torch.no_grad()
def sample_ddim(model, scheduler, shape=(1, 3, 32, 32), num_steps=50):
    """DDIM 加速采样"""
    model.eval()
    
    # 设置时间步
    scheduler.set_timesteps(num_steps)
    
    # 从纯噪声开始
    x = torch.randn(shape).to(model.device)
    
    for t in scheduler.timesteps:
        # 预测噪声
        noise_pred = model(x, t)
        
        # DDIM 采样一步
        x = scheduler.step(noise_pred, t, x)
    
    return x
```

### 8.3 与 DDPM 对比

```python
# DDPM 采样 (1000 步)

# 耗时：~100 秒

# 质量损失：FID 9.4 → 12.5
```

## 深入理解

### 9.1 为什么可以跳步？

```
直观理解:

DDPM:
x₀ → x₁ → x₂ → ... → x_T
每一步只去除一点噪声

DDIM:
x₀ → x₂₀ → x₄₀ → ... → x_T
每一步去除更多噪声

关键:
- 训练时模型见过所有噪声水平
- 所以可以在任意噪声水平去噪
- 不需要严格按顺序
```

### 9.2 确定性采样的好处

```
1. 可复现性
   - 相同种子 → 相同结果
   - 便于调试和对比

2. 可优化性
   - 可以直接优化 x_T
   - 获得更好的生成结果

3. 可解释性
   - x_T 与 x_0 一一对应
   - 隐空间结构更清晰

4. 图像修复
   - 可以固定已知区域
   - 只优化未知区域
```

### 9.3 质量 vs 速度的权衡

```
步数选择建议:

| 应用场景 | 推荐步数 | FID | 耗时 |
|---------|---------|-----|------|
| 高质量生成 | 100-200 | 10-11 | 10-20 秒 |
| 日常使用 | 50 | 12-13 | 5 秒 |
| 快速预览 | 20 | 15-16 | 2 秒 |
| 实时交互 | 10 | 20-22 | 1 秒 |

建议:
- 先用少步数预览
- 满意后再用多步数生成
```

## 与 DDPM 的关系

### 10.1 理论关系

```
DDPM 是 DDIM 的特例:

当 σ_t = √((1-ᾱ_{t-1})/(1-ᾱ_t)) × √(1-ᾱ_t/ᾱ_{t-1}) × β_t 时:
DDIM → DDPM

当 σ_t = 0 时:
DDIM → 确定性 DDIM

DDIM 是一个统一框架!
```

### 10.2 实践关系

```
训练:
- DDPM 和 DDIM 使用相同的训练方法
- DDPM 训练好的模型可以直接用于 DDIM 采样

采样:
- DDPM: 必须 1000 步，随机采样
- DDIM: 可以 10-100 步，确定性采样

升级:
- 有了 DDPM 模型，自动获得 DDIM 能力
- 无需重新训练
```

### 10.3 性能对比

| 特性 | DDPM | DDIM |
|------|------|------|
| 训练方法 | 相同 | 相同 |
| 采样步数 | 1000 | 10-100 |
| 采样速度 | 慢 | 快 (10-100x) |
| 生成质量 | 最好 | 略差但可接受 |
| 可复现性 | 低 | 高 |
| 图像修复 | 困难 | 支持 |

## 加速采样方法

### 11.1 时间步选择

```
均匀采样:
timesteps = [1, 20, 40, 60, ..., 1000]

非均匀采样 (推荐):
timesteps = [1, 5, 10, 20, 30, 50, 70, ..., 1000]

原理:
- 早期噪声大，需要更密集的步骤
- 后期噪声小，可以跳更大步
```

### 11.2 蒸馏加速

```
知识蒸馏:
训练一个学生模型，直接用更少的步数

方法:
1. 用 DDIM 生成 (x_T, x_0) 对
2. 训练学生模型直接从 x_T → x_0
3. 只需 1-4 步

代表工作:
- SD Turbo (1 步)
- LCM (4 步)
```

### 11.3 其他加速方法

| 方法 | 原理 | 加速比 |
|------|------|--------|
| DDIM | 跳步采样 | 10-50x |
| 蒸馏 | 知识蒸馏 | 100-1000x |
| ODE 求解器 | 高阶求解器 | 2-5x |
| 并行采样 | 同时采样多步 | 2-4x |

1. **非马尔可夫扩散**: 保持边缘分布，改变条件分布
2. **统一框架**: DDPM 是 DDIM 的特例
3. **加速采样**: 从 1000 步减少到 10-100 步
4. **确定性采样**: 可复现、可优化、可解释

- **重要性**: 解决了 Diffusion 采样的核心瓶颈
- **影响力**: 8,000+ 引用，被广泛采用
- **实用性**: 所有现代 Diffusion 模型都支持 DDIM 采样

### 学习价值

- **理论基础**: 理解扩散过程的本质
- **实践价值**: 日常使用 DDIM 而非 DDPM
