## 数学原理

### 4.1 噪声调度

```
噪声调度 σ(t):

DDPM:
β_t 线性增加
σ_t = √(1-ᾱ_t)

EDM:
σ(t) = exp(ρt)
ρ = 7 (推荐)

其中 t ∈ [-3, 3]

优势:
- 对数分布
- 覆盖更广范围
- 更好采样
```

### 4.2 损失加权

```
损失加权 λ(σ):

DDPM:
均匀加权
λ(σ) = 1

EDM:
λ(σ) = (σ² + σ_data²) / (σ² + σ_min²)²

其中:
- σ_data: 数据标准差
- σ_min: 最小噪声

直观:
- 中等噪声权重高
- 极低/极高噪声权重低
```

### 4.3 二阶采样器

```
Heun 采样器 (二阶):

Predictor:
x' = x + h·v(x, t)

Corrector:
x'' = x + h/2 · [v(x, t) + v(x', t+h)]

其中:
- h: 步长
- v: 向量场

优势:
- 比 Euler 精确
- 更少步数
- 质量更好
```

## 训练方法

### 6.1 训练算法

```
Algorithm 1: EDM 训练

Input: 数据集 {x⁽¹⁾, ..., x⁽ᴺ⁾}
Initialize: UNet 参数 θ

repeat
    for each x⁽ⁱ⁾:
        # 采样噪声水平
        σ ~ p(σ)  # 对数均匀
        
        # 采样噪声
        ε ~ N(0, I)
        
        # 加噪
        x_σ = x⁽ⁱ⁾ + σ ε
        
        # 预测去噪
        D_θ(x_σ, σ)
        
        # 损失 (加权)
        L = λ(σ) · ||D_θ(x_σ, σ) - x⁽ⁱ⁾||²
    
    更新 θ
until 收敛
```

```
噪声水平分布:

p_log_uniform(σ):
```

## 代码实现

### 8.1 噪声调度

```python
class EDMNoiseScheduler:
    def __init__(self, sigma_min=0.002, sigma_max=80, rho=7):
        self.sigma_min = sigma_min
        self.sigma_max = sigma_max
        self.rho = rho
    
    def sigma(self, t):
        """噪声调度 σ(t)"""
        # t in [-3, 3]
        return torch.exp(self.rho * t)
    
    def sample_sigma(self, batch_size, device):
        """采样噪声水平"""
        # 对数均匀采样
        log_sigma = torch.rand(batch_size, device=device) * \
                    (torch.log(torch.tensor(self.sigma_max)) - \
                     torch.log(torch.tensor(self.sigma_min))) + \
                    torch.log(torch.tensor(self.sigma_min))
        return torch.exp(log_sigma)
```

### 8.2 损失加权

```python
def edm_loss_weight(sigma, sigma_data=0.5, sigma_min=0.002):
    """EDM 损失加权"""
    return (sigma**2 + sigma_data**2) / (sigma**2 + sigma_min**2)**2
```

### 8.3 训练循环

```python
def train_edm(model, dataloader, optimizer, device, epochs=10):
    model.train()
    scheduler = EDMNoiseScheduler()
    
    for epoch in range(epochs):
        total_loss = 0
        
        for batch_idx, (x0, _) in enumerate(dataloader):
            x0 = x0.to(device)
            optimizer.zero_grad()
            
            # 采样噪声水平
            sigma = scheduler.sample_sigma(x0.shape[0], device)
            
            # 采样噪声
            noise = torch.randn_like(x0)
            
            # 加噪
            x_sigma = x0 + sigma[:, None, None, None] * noise
            
            # 预测去噪
            pred = model(x_sigma, sigma)
            
            # 损失 (加权)
            weight = edm_loss_weight(sigma)
            loss = (weight[:, None, None, None] * (pred - x0)**2).mean()
            
            # 反向传播
            loss.backward()
            optimizer.step()
            
            total_loss += loss.item()
        
        avg_loss = total_loss / len(dataloader)
        print(f'Epoch {epoch+1}: Loss = {avg_loss:.4f}')
```

### 8.4 Heun 采样器

```python
@torch.no_grad()
def sample_heun(model, scheduler, shape=(1, 3, 32, 32), num_steps=128):
    """Heun 二阶采样器"""
    model.eval()
    
    # 从噪声开始
    x = torch.randn(shape).to(model.device)
    
    # 时间步
    sigmas = scheduler.sigma(torch.linspace(-3, 3, num_steps))
    
    for i in range(num_steps - 1):
        sigma = sigmas[i]
        sigma_next = sigmas[i + 1]
        
        # Predictor (Euler)
        d = (x - model(x, sigma)) / sigma
        x_prime = x + (sigma_next - sigma) * d
        
        # Corrector (Heun)
        d_prime = (x_prime - model(x_prime, sigma_next)) / sigma_next
        x = x + (sigma_next - sigma) * 0.5 * (d + d_prime)
    
    return x
```

## 深入理解

### 9.1 为什么对数噪声调度好？

```
直觉解释:

线性调度:
- 均匀分布
- 覆盖范围有限

对数调度:
- 对数均匀
- 覆盖更广范围
- 从 0.002 到 80

好处:
- 更好覆盖所有噪声水平
- 训练更稳定
- 采样质量更好
```

### 9.2 为什么需要损失加权？

```
损失加权的原因:

不同噪声水平:
- 低噪声：容易去噪
- 中噪声：最难
- 高噪声：几乎纯噪声

均匀加权问题:
- 低/高噪声主导
- 中等噪声学习不足

EDM 加权:
- 中等噪声权重高
- 平衡学习
- 更好收敛
```

### 9.3 二阶采样器的优势

```
Euler vs Heun:

Euler (一阶):
x_{n+1} = x_n + h·v(x_n)
- 简单
- 误差 O(h²)

Heun (二阶):
x_{n+1} = x_n + h/2·[v(x_n) + v(x_{n+1}')]
- 更精确
- 误差 O(h³)
- 更少步数

结果:
- Heun 用一半步数达到相同质量
```
