## 数学原理

### 4.1 双编码器融合

```
文本编码:

编码器 1 (OpenCLIP):
c₁ = τ₁(y) ∈ R^(77×1280)

编码器 2 (OpenAI CLIP):
c₂ = τ₂(y) ∈ R^(77×768)

融合:
c = [c₁; c₂] ∈ R^(77×2048)

投影:
c' = W·c ∈ R^(77×1280)

其中 W 是可学习投影矩阵
```

### 4.2 级联扩散

```
两阶段生成:

阶段 1 (Base):
z₀^base ~ p(z₀|y)
x₀^base = D(z₀^base)

阶段 2 (Refiner):
z₀^refine ~ p(z₀|x₀^base, y)
x₀^final = D(z₀^refine)

其中:
- Base: 从文本生成
- Refiner: 从 Base 输出 + 文本生成
```

### 4.3 微分方程视角

```
连续时间视角:

Base 模型:
dx = [f(x,t) - g(t)²∇log p(x|y)]dt + g(t)dw̄

Refiner 模型:
dx = [f(x,t) - g(t)²∇log p(x|x_base,y)]dt + g(t)dw̄

关键:
- Refiner 的条件包含 x_base
- 细化已有生成
- 添加细节
```

## 训练方法

### 6.1 训练算法

```
Algorithm 1: SDXL 训练

Input: 图像 + 文本数据集 {(x⁽¹⁾, y⁽¹⁾), ...}
Initialize: Base UNet, Refiner UNet (可选)

# 阶段 1: 训练 Base 模型
repeat
    for each (x⁽ⁱ⁾, y⁽ⁱ⁾):
        # 双文本编码
        c₁ = OpenCLIP(y⁽ⁱ⁾)
        c₂ = CLIP(y⁽ⁱ⁾)
        c = [c₁; c₂]
        
        # VAE 编码
        z₀ = VAE.Encode(x⁽ⁱ⁾)
        
        # 加噪
        t ~ Uniform(1, T)
        ε ~ N(0, I)
        z_t = √ᾱ_t z₀ + √(1-ᾱ_t) ε
        
        # 预测噪声
        ε_θ = BaseUNet(z_t, t, c)
        
        # 损失
        L = ||ε - ε_θ||²
    
    更新 Base UNet 参数
until 收敛

# 阶段 2: 训练 Refiner (可选)
repeat
    for each (x⁽ⁱ⁾, y⁽ⁱ⁾):
        # Base 模型推理
        z₀^base = BaseUNet.Sample(y⁽ⁱ⁾)
        x₀^base = VAE.Decode(z₀^base)
        
        # Refiner 训练
        # ... 类似 Base 训练
    更新 Refiner 参数
until 收敛
```

### 8.1 双文本编码器

```python
class DualTextEncoder(nn.Module):
    def __init__(self):
        super().__init__()
        
        # OpenCLIP ViT-bigG
        self.openclip = open_clip.create_model('ViT-bigG-14', pretrained='laion2b_s39b_b160k')
        
        # OpenAI CLIP ViT-L
        self.clip = CLIPTextModel.from_pretrained("openai/clip-vit-large-patch14")
        
        # 投影层
        self.projection = nn.Linear(2048, 1280)
    
    def forward(self, text):
        # OpenCLIP 编码
        c1 = self.openclip.encode_text(text)  # (B, 77, 1280)
        
        # CLIP 编码
        c2 = self.clip(text).last_hidden_state  # (B, 77, 768)
        
        # 拼接
        c = torch.cat([c1, c2], dim=-1)  # (B, 77, 2048)
        
        # 投影
        c = self.projection(c)  # (B, 77, 1280)
        
        return c
```
