## 数学原理

### 4.1 MAE 风格预训练

```
Masked Autoencoder 损失:

输入: x
Mask: M (二元掩码)
可见部分: x_visible = (1-M) ⊙ x
Masked 部分: x_masked = M ⊙ x

编码器:
z = Encoder(x_visible)

解码器:
x_recon = Decoder(z)

损失:
L = ||x_masked - x_recon_masked||²

只重建 masked 部分
```

### 4.2 自编码微调

```
Autoencoder 微调损失:

输入：完整图像 x
编码：z = Encoder(x)
解码：x' = Decoder(z)

损失:
L = L_recon + λ L_regularization

L_recon = ||x - x'||²
L_regularization: 保持表示质量
```

### 4.3 联合训练

```
RAE + DiT 联合损失:

L = L_VAE + L_Diffusion

L_VAE = 重建损失 + KL 散度
L_Diffusion = Diffusion 损失

联合优化:
- 共享编码器
- 端到端
- 更好配合
```

## 训练方法

### 6.1 预训练算法

```
Algorithm 1: RAE 预训练

Input: 大规模图像数据集
Initialize: ViT Encoder, ViT Decoder

repeat
    for each x:
        # 随机 Mask
        M ~ Bernoulli(0.75)  # 75% mask
        
        # 可见部分
        x_visible = (1-M) ⊙ x
        
        # 编码
        z = Encoder(x_visible)
        
        # 解码
        x_recon = Decoder(z)
        
        # 只重建 masked 部分
        L = ||M ⊙ x - M ⊙ x_recon||²
    
    更新 Encoder, Decoder
until 收敛
```

### 6.2 微调算法

```
Algorithm 2: RAE 微调

Input: 图像数据集，预训练权重
Initialize: 载入预训练权重

repeat
    for each x:
        # 完整编码 - 解码
        z = Encoder(x)
        x_recon = Decoder(z)
        
        # 完整重建损失
        L = ||x - x_recon||² + λ L_KL
    
    更新 Encoder, Decoder
until 收敛
```

## 代码实现

### 8.1 RAE 模型

```python
class RAE(nn.Module):
    def __init__(self, patch_size=16, embed_dim=768):
        super().__init__()
        
        # Patch embedding
        self.patch_embed = PatchEmbed(patch_size, embed_dim)
        
        # Encoder (ViT)
        self.encoder = ViTEncoder(depth=12, embed_dim=embed_dim)
        
        # Decoder (ViT)
        self.decoder = ViTDecoder(depth=8, embed_dim=embed_dim)
        
        # Mask token
        self.mask_token = nn.Parameter(torch.zeros(1, 1, embed_dim))
    
    def forward(self, x, mask_ratio=0.75):
        # Patchify
        patches = self.patch_embed(x)
        
        # Random mask
        mask = torch.rand(patches.shape[:2]) < mask_ratio
        patches_visible = patches[~mask]
        
        # Encode
        z = self.encoder(patches_visible)
        
        # Add mask tokens
        z_full = self.add_mask_tokens(z, mask)
        
        # Decode
        x_recon = self.decoder(z_full)
        
        return x_recon, mask
```

### 8.2 训练损失

```python
def rae_pretrain_loss(x, x_recon, mask):
    """RAE 预训练损失 (只计算 masked 部分)"""
    # 只重建 masked patches
    loss = (mask * (x - x_recon)**2).sum() / mask.sum()
    return loss

def rae_finetune_loss(x, x_recon, mu, logvar):
    """RAE 微调损失"""
    # 重建损失
    recon_loss = F.mse_loss(x, x_recon)
    
    # KL 损失
    kl_loss = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp())
    
    return recon_loss + 1e-6 * kl_loss
```

## 深入理解

### 9.1 为什么自监督预训练有效？

```
直觉解释:

自监督学习:
- 无需标注
- 大规模数据
- 学习通用表示

MAE 风格:
- Mask + 重建
- 学习上下文
- 紧凑表示

好处:
- 更好的初始化
- 更快收敛
- 更好质量
```

### 9.2 RAE vs VAVAE

```
对比:

VAVAE:
- 外部基础模型 (DINOv2)
- 对齐损失
- 依赖外部模型

RAE:
- 自学习基础模型
- MAE 预训练
- 自给自足

选择:
- VAVAE: 有 DINOv2 时
- RAE: 从头训练时
```
