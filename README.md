## 📚 资料库简介

这是一个系统化的 AIGC（AI Generated Content）学习资料库，专注于图像生成方向，包含论文、技术文档、代码实践指南等资源。

**🎉 新增**: 具身智能 (Embodied AI) 专题 - 包含 2025-2026 年最新论文、代码项目和公司动态！

## 🚀 快速开始

### 新手学习路线

1. **第 1 步**: 阅读 `01-基础理论/Diffusion 学习路线 - 关键论文汇总.md`
2. **第 2 步**: 按顺序阅读核心论文（VAE → DDPM → DDIM → LDM）
3. **第 3 步**: 参考 `06-代码项目/代码实践指南.md` 动手实践
4. **第 4 步**: 使用 `12-笔记整理/论文笔记模板.md` 做笔记

### 推荐阅读顺序

```
VAE (2013)
  ↓
DDPM (2020) → DDIM (2020)
  ↓
LDM/Stable Diffusion (2021)
  ↓
ControlNet (2023) → DiT (2022) → LoRA (2021)
  ↓

```
