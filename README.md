## 📋 建议搜集内容清单

### ✅ 已有规划

### 🔲 建议补充搜集

| 类别 | 具体内容 | 优先级 |
|------|---------|--------|
| **专利** | 各公司 AIGC 相关专利申请 | 中 |
| **融资信息** | 创业公司融资轮次、金额、投资方 | 中 |
| **产品信息** | 各公司产品的定价、功能对比 | 高 |
| **API 文档** | 各大模型 API 使用文档、价格 | 高 |
| **基准测试** | 各模型在主流 Benchmark 上的表现 | 高 |
| **开源许可证** | 各开源模型的许可证类型（商用限制） | 高 |
| **技术博客** | 公司技术团队的博客文章 | 中 |
| **招聘信息** | 各公司招聘方向（反映技术重点） | 低 |
| **合作伙伴** | 公司间的合作关系、生态联盟 | 中 |
| **负面新闻** | 技术争议、安全事故、诉讼 | 中 |
| **监管动态** | 各国 AI 监管政策更新 | 高 |
| **开源社区** | GitHub Trending、HuggingFace 热门 | 中 |
| **播客/访谈** | 创始人/研究者访谈 | 低 |
| **竞品分析** | 同类产品的详细对比 | 高 |

## 📝 使用建议

### 文件命名规范
```
[日期]_[类型]_[标题].[格式]
例：2026-04-03_论文_Sora 高保真视频生成.pdf
例：2026-04-03_代码_StableDiffusion 复现.zip
例：2026-04-03_报告_中国 AIGC 行业分析.pdf
```

### 笔记模板
在 `12-笔记整理/01-论文笔记/` 中使用统一模板：
```markdown
# 论文标题

## 基本信息
- 作者：
- 机构：
- 日期：
- 链接：

## 核心贡献
1. ...
2. ...

## 方法概述
...

## 实验结果
...

## 个人思考
...

## 相关资源
- 代码：
- 数据：
```

### 定期维护
- **每周**: 整理新搜集的资料到对应目录
- **每月**: 更新笔记整理，删除重复/过时内容
