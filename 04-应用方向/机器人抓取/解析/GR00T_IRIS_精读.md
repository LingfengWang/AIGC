## GR00T: Generalist Robot 00 Foundation Model (2024)

### 基本信息

| 项目 | 内容 |
|------|------|
| 论文 | GR00T: Generalist Robot 00 Foundation Model |
| 作者 | NVIDIA |
| arXiv | 2405.03533 |
| 时间 | 2024年5月 |

---

### 核心问题

1. **数据效率**：机器人数据稀缺
2. **泛化能力**：跨机器人、跨任务
3. **通用性**：单一模型完成多种任务

---

### 核心创新

**三大特性**：

1. **通用性** (Generalist)
   - 单一模型处理多种机器人
   - 多种任务类型

2. **互联网规模** (Web-Scale)
   - 利用互联网视频预训练
   - 视觉-语言先验

3. **多模态输入** (Multimodal)
   - 图像 + 视频 + 语言 + 动作

---

### 架构

```
输入:
  - 图像/点云 O
  - 语言指令 L
  - 动作历史 A
  
处理:
  - 视觉编码器 (预训练)
  - 语言编码器 (预训练)
  - 扩散策略 (动作生成)
  
输出:
  - 动作序列 A
```

---

### 技术特点

#### 1. 扩散策略

```python
class GR00TPolicy:
    def __init__(self):
        self.visual_encoder = ViT()
        self.language_encoder = Llama()
        self.action_diffusion = DiffusionUNet()
    
    def forward(self, obs, lang):
        # 1. 编码
        v = self.visual_encoder(obs)
        l = self.language_encoder(lang)
        
        # 2. 融合
        h = fusion(v, l)
        
        # 3. 扩散生成动作
        a = self.action_diffusion(h)
        
        return a
```

#### 2. 动作 chunking

- 生成 **N 个未来动作** 而非单步
- 提高控制频率

## IRIS: Implicit Reinforcement with Self-supervised Learning (2023)

### 基本信息

| 项目 | 内容 |
|------|------|
| 论文 | IRIS: Implicit Reinforcement with Self-supervised Learning |
| 作者 | Stanford |
| arXiv | 2305.18654 |

---

### 核心创新

#### 1. 隐式强化学习

**与传统 RL 对比**：

| 传统 RL | IRIS |
|---------|------|
| 显式奖励函数 | 无需显式奖励 |
| 稀疏奖励处理困难 | 自监督学习 |
| 需要大量交互 | 样本高效 |

#### 2. 自监督任务

```python
# IRIS 自监督任务
tasks = [
    "重建观测",
    "对比学习",
    "动作预测",
    "因果推断"
]
```

#### 3. 隐式世界模型

- 学习潜在动力学
- 不需要像素重建
- 更高效

---

### 对比总结

### GR00T vs IRIS

| 方面 | GR00T | IRIS |
|------|-------|------|
| 数据来源 | 互联网视频 + 机器人 | 机器人数据 |
| 泛化方式 | 预训练 | 自监督 |
| 动作生成 | 扩散 | 策略网络 |
| 适用场景 | 通用 | 特定任务 |

---

### 世界模型选择指南

| 场景 | 推荐方法 |
|------|---------|
| **样本效率优先** | Dreamer 系列 |

| **资源有限** | IRIS |
| **通用机器人** | GR00T |

## 核心方法论总结

### 1. 数据来源

```
机器人数据 → 昂贵但精确
   ↓
互联网视频 → 便宜但间接
   ↓
合成数据 → 可控但简化
```

### 2. 学习范式

```
监督学习 → 模仿人类演示
   ↓
自监督学习 → 重建/对比
   ↓
强化学习 → 环境交互
   ↓
世界模型 → 想象规划
```

### 3. 当前SOTA组合

```
互联网视频预训练 (泛化)
    ↓
世界模型 (规划)
    ↓
VLA (执行)
    ↓
反馈 (持续学习)
```
