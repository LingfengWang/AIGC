### 1. VAE (2013)

**📄 论文**: Auto-Encoding Variational Bayes  
**👤 作者**: Kingma & Welling (University of Amsterdam)  
**📅 发表**: ICLR 2014  
**📊 引用**: 25,000+  
**💾 文件**: `05_VAE_2013_Kingma.pdf` (3.7 MB)

#### 核心问题
如何学习数据的**连续隐变量表示**，并能够从中生成新样本？

#### 核心贡献
1. **重参数化技巧**
   ```
   传统采样：z ~ N(μ, σ²)  ← 不可导
   重参数化：ε ~ N(0, 1), z = μ + σ·ε  ← 可导!
   ```

2. **ELBO 损失函数**
   ```
   L = E_q[log p(x|z)] - KL(q(z|x) || p(z))
      重构损失           正则化项
   ```

#### 为什么重要
- Diffusion 可以看作"多个 VAE 串联"
- 理解隐空间 (latent space) 概念的基础
- 重参数化技巧被广泛使用

#### 学习建议
- **重点理解**: 重参数化技巧、ELBO 推导
- **代码实践**: 实现一个简单的 VAE
- **用时**: 4-6 小时

### 2. DDPM (2020)

**📄 论文**: Denoising Diffusion Probabilistic Models  
**👤 作者**: Jonathan Ho et al. (UC Berkeley)  
**📅 发表**: NeurIPS 2020  
**📊 引用**: 15,000+  
**💾 文件**: `01_DDPM_2020_Ho.pdf` (9.8 MB)

#### 核心问题
如何生成高质量图像，同时避免 GAN 的训练不稳定问题？

#### 核心贡献
1. **扩散概率模型框架**
   - 前向过程：逐步加噪 (固定)
   - 反向过程：逐步去噪 (学习)

2. **预测噪声而非原图** (关键洞察!)
   ```
   损失函数：L = E[||ε - ε_θ(x_t, t)||²]
   ```

3. **任意时刻直接采样**
   ```
   q(x_t | x_0) = N(x_t; √ᾱ_t x_0, (1-ᾱ_t)I)
   ```

#### 为什么重要
- Diffusion 领域的"ImageNet 时刻"
- 训练稳定，不会 mode collapse
- 生成质量媲美 GAN

#### 学习建议
- **重点理解**: 为什么预测噪声、噪声调度
- **代码实践**: 实现 DDPM 训练和采样
- **用时**: 10-12 小时

### 3. DDIM (2020)

**📄 论文**: Denoising Diffusion Implicit Models  
**👤 作者**: Jiaming Song et al. (Stanford, UC Berkeley)  
**📅 发表**: ICLR 2021  
**📊 引用**: 8,000+  
**💾 文件**: `02_DDIM_2020_Song.pdf` (10 MB)

#### 核心问题
DDPM 需要 1000 步采样，太慢了！能否减少采样步数？

#### 核心贡献
1. **非马尔可夫扩散过程**
   ```
   DDPM: x_t → x_{t-1} → ... → x_0  (马尔可夫)
   DDIM: x_t → x_{t-Δ} → ... → x_0  (可以跳步!)
   ```

2. **确定性采样**
   - 相同起点 → 相同结果
   - 便于调试和复现

#### 为什么重要
- 从 1000 步加速到 50-100 步
- 确定性采样便于应用
- 理论贡献：统一框架

#### 学习建议
- **重点理解**: 跳步采样原理
- **代码实践**: 实现 DDIM 采样器
- **用时**: 6-8 小时

### 4. Score SDE (2020)

**📄 论文**: Score-Based Generative Modeling through Stochastic Differential Equations  
**👤 作者**: Yang Song et al. (Stanford, Google Brain)  
**📅 发表**: ICLR 2021  
**📊 引用**: 5,000+  
**💾 文件**: `16_ScoreSDE_2020_Song.pdf` (26 MB)

#### 核心问题
如何统一 Score Matching 和 Diffusion Models 的理论框架？

#### 核心贡献
1. **分数匹配 (Score Matching)**
   ```
   学习目标：s_θ(x) ≈ ∇_x log p(x)
   ```

2. **SDE 框架**
   ```
   前向 SDE: dx = f(x,t)dt + g(t)dw
   反向 SDE: dx = [f(x,t) - g(t)² ∇_x log p_t(x)]dt + g(t)dw̄
   ```

3. **概率流 ODE**
   - 确定性连续时间生成过程

#### 为什么重要
- 理论统一：多个生成模型框架
- SDE 视角：新的理论工具
- 后续影响：很多工作基于此

#### 学习建议
- **重点理解**: 分数函数、SDE 基础
- **前置知识**: 随机过程基础
- **用时**: 8-10 小时

### 6. LDM/Stable Diffusion (2021)

**📄 论文**: High-Resolution Image Synthesis with Latent Diffusion Models  
**👤 作者**: Robin Rombach et al. (CompVis, Stability AI)  
**📅 发表**: CVPR 2022  
**📊 引用**: 10,000+  
**💾 文件**: `03_LDM_2021_Rombach.pdf` (37 MB)

#### 核心问题
DDPM 在像素空间训练，计算量太大。如何生成高分辨率图像？

#### 核心贡献
1. **隐空间扩散**
   ```
   像素空间：512×512×3 = 786,432 维
   隐空间：64×64×4 = 16,384 维  ← 压缩 48 倍!
   ```

2. **两阶段训练**
   - 阶段 1: 训练 VAE
   - 阶段 2: 隐空间训练 Diffusion

3. **Cross-Attention 条件机制**
   ```
   文本 → CLIP → 文本嵌入
   图像隐变量 + 文本嵌入 → Cross-Attention
   ```

#### 为什么重要
- 计算效率：训练成本降低 48 倍
- 开源生态：引爆社区创新
- 商业化：AIGC 进入大众视野

#### 学习建议
- **重点理解**: 隐空间压缩、Cross-Attention
- **代码实践**: 使用 Diffusers 运行 SD
- **用时**: 10-12 小时

### 9. LoRA (2021)

**📄 论文**: LoRA: Low-Rank Adaptation of Large Language Models  
**👤 作者**: Edward J. Hu et al. (Microsoft)  
**📅 发表**: ICLR 2022  
**📊 引用**: 5,000+  
**💾 文件**: `07_LoRA_2021_Hu.pdf` (1.5 MB)

#### 核心问题
大模型全参数微调成本太高。能否只微调少量参数？

#### 核心贡献
1. **低秩分解**
   ```
   原始权重更新：ΔW (d×k)
   LoRA 分解：ΔW = B·A  (B: d×r, A: r×k, r << d)
   r=8 时，参数量减少 1000 倍!
   ```

2. **冻结 + 旁路**
   ```
   冻结原始模型权重 W
   训练 LoRA 权重 A, B
   推理时：W' = W + B·A
   ```

3. **无推理延迟**
   - 训练完成后可合并
   - 推理时无额外开销

#### 为什么重要
- 高效微调：参数量减少 1000 倍
- 2023 年应用于 SD
- 催生数十万 LoRA 模型

#### 学习建议
- **重点理解**: 低秩分解原理
- **代码实践**: 训练 LoRA
- **用时**: 6-8 小时

### 13. Rectified Flow (2022)

**📄 论文**: Flow Straight and Fast: Learning to Generate and Transfer Data with Rectified Flow  
**👤 作者**: Xingchao Liu et al. (UT Austin)  
**📅 发表**: ICLR 2023  
**📊 引用**: 1,000+  
**💾 文件**: `08_RectifiedFlow_2022_Liu.pdf` (17 MB)

#### 核心问题
Diffusion 的噪声路径是弯曲的，采样效率低。能否让生成路径更直、更快？

#### 核心贡献
1. **直线路径**
   ```
   Diffusion: x_0 → 弯曲路径 → x_1
   Rectified Flow: x_0 → 直线路径 → x_1
   ```

2. **常微分方程 (ODE)**
   ```
   dx/dt = v(x, t)  ← 学习速度场
   ```

3. **更少采样步数**
   ```
   Diffusion: 100-1000 步
   Rectified Flow: 10-50 步即可
   ```

#### 为什么重要
- 效率：采样速度快 10-20 倍
- 质量：生成质量相当或更好
- 采用：SD3、FLUX 都采用

#### 学习建议
- **重点理解**: 流模型 vs 扩散模型
- **对比学习**: Rectified Flow vs DDPM
- **用时**: 6-8 小时

## 📊 总结对比

### 时间线

```
2013: VAE
  ↓
2020: DDPM → DDIM → Score SDE
  ↓
2021: Diffusion Beat GANs → LDM
  ↓
2022: DiT → LoRA(应用于 SD) → EDM → Rectified Flow → Flow Matching
  ↓
2023: ControlNet → SDXL
  ↓
2024: MMDiT (SD3)
  ↓
2025: VAVAE → RAE → JiT → PixelDiT
```

### 技术演进

| 阶段 | 代表技术 | 特点 |
|------|---------|------|
| 基础理论 | VAE | 隐空间概念 |
| Diffusion 奠基 | DDPM, DDIM | 加噪 - 去噪范式 |
| 商业化 | LDM/SD | 隐空间扩散 |
| 架构演进 | DiT, MMDiT | Transformer 替代 UNet |
| 效率优化 | Rectified Flow, Flow Matching | 流模型，一步生成 |
| 前沿探索 | VAVAE, RAE, JiT, PixelDiT | 编码器改进，像素空间 |
