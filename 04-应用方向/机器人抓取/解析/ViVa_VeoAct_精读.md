## ViVa: Video-Generative Value Model (2026)

### 基本信息

| 项目 | 内容 |
|------|------|
| 论文 | ViVa: A Video-Generative Value Model for Robot Reinforcement Learning |
| 作者 | Jindi Lv et al. |
| arXiv | 2604.01901 |
| 时间 | 2026年4月 |

---

### 核心问题

1. **VLA 模型的局限性**：
   - 依赖大规模机器人数据
   - 泛化能力有限
   - 难以理解物理规律

2. **视频生成的优势**：
   - 互联网大规模视频
   - 包含丰富的物理知识

---

### 核心创新

**架构**：

```
┌─────────────────────────────────────────────────────────────┐
│                      ViVa 架构                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  输入: 观测 o_t                                             │
│    ↓                                                         │
│  视频生成器 G(o_t, a_t) → 预测未来帧 o_{t+1:H}             │
│    ↓                                                         │
│  价值模型 V(o_{t+1:H}) → 估计累积奖励                       │
│    ↓                                                         │
│  策略优化 π(a_t | o_t) → 生成动作                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

---

### 关键技术

#### 1. 视频-价值联合学习

```python
class ViVa:
    def __init__(self):
        self.video_gen = VideoGenerator()   # 视频生成
        self.value_model = ValueModel()     # 价值评估
    
    def forward(self, obs, action):
        # 1. 生成未来视频
        future_frames = self.video_gen(obs, action)
        
        # 2. 评估价值
        value = self.value_model(future_frames)
        
        return value
```

#### 2. 真实视频预训练

- 使用 **真实机器人操作视频** 预训练
- 学习物理规律和因果关系
- 理解物体交互

#### 3. 分层价值评估

```
价值 = 立即奖励 + 未来价值
     = R(o_t, a_t) + γ * V(o_{t+1})
```

### 实验结果

| 方法 | 成功率 | 泛化能力 |
|------|--------|---------|
| 无模型 RL | 45% | 低 |

### 应用场景

- **抓取规划**：预测抓取后的状态
- **碰撞避免**：预测物体移动
- **放置规划**：预测放置稳定性

## Veo-Act: 前沿视频模型用于机器人 (2026)

### 基本信息

| 项目 | 内容 |
|------|------|
| 论文 | Veo-Act: How Far Can Frontier Video Models Advance Generalizable Robot Manipulation? |
| 作者 | Zhongru Zhang et al. |
| arXiv | 2604.07999 |
| 时间 | 2026年4月 |

---

### 核心问题

**关键挑战**：视频模型能否直接用于机器人控制？

- 视频 ≠ 动作
- 泛化 vs 精确控制
- 零样本能力

---

### 核心创新

#### 1. 视频-动作映射

```
视频模型 (Veo) + 动作空间 → 机器人控制
     ↓
  视频帧 → 动作预测
```

#### 2. 零样本泛化测试

| 测试场景 | 结果 |
|----------|------|
| 训练物体 | 85% |
| 新物体 | 72% |
| 新背景 | 68% |
| 新相机 | 65% |

#### 3. 数据效率

| 方法 | 所需数据量 |
|------|-----------|
| VLA (RT-2) | 100k+ episodes |
| Veo-Act | 0 (零样本) |
| Veo-Act + 微调 | 1k episodes |

---

### 对比分析

### ViVa vs Veo-Act

| 方面 | ViVa | Veo-Act |
|------|------|---------|
| 核心思想 | 视频生成+价值评估 | 视频模型直接动作 |
| 训练数据 | 机器人视频 | 互联网视频 |
| 样本效率 | 高 | 零样本 |
| 精确控制 | 中 | 中 |
| 泛化能力 | 高 | 很高 |

---

### 与 VLA 对比

| 指标 | VLA (RT-2) | ViVa | Veo-Act |
|------|------------|------|---------|
| 泛化能力 | 中 | 高 | **很高** |
| 数据需求 | 高 | 中 | **零** |
| 推理速度 | 快 | 中 | 中 |
| 物理理解 | 中 | **高** | **高** |

---

### 关键洞察

#### 1. 为什么视频模型有效？

- **物理先验**：视频包含物体如何移动、交互
- **因果关系**：动作→结果的模式
- **多样性**：互联网视频覆盖各种场景

#### 2. 局限性

- **细节丢失**：视频 → 精确动作有差距
- **控制频率**：难以实时控制
- **物理交互**：无法直接获取力/触觉

#### 3. 未来方向

```
视频模型 → 世界模型 → VLA
   ↓            ↓          ↓
 泛化       物理理解    精确控制
```

## 面试要点

### Q1: 视频生成模型与VLA的区别？

| VLA | 视频生成模型 |
|-----|-------------|
| 直接输出动作 | 预测未来帧 |
| 需要机器人数据 | 可用互联网视频 |
| 精确控制 | 泛化更好 |

### Q2: 如何从视频生成过渡到动作？

1. **直接映射**：视频帧 → 动作
2. **价值评估**：选择动作后评估
3. **逆模型**：预测需要什么动作

### Q3: 零样本泛化的意义？

- 减少数据收集成本
- 快速适应新任务
- 泛化到未见过的场景

### Q4: 当前主要挑战？

1. 视频-动作差距
2. 实时控制频率
3. 物理交互理解

### Q5: 实际应用价值？

- 新场景快速部署
- 减少人工数据标注
- 辅助 VLA 训练
